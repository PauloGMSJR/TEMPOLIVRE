# TaskBro – Produtividade sem frescura

Interface conceitual e interativa do aplicativo TaskBro, um app de produtividade com foco no público masculino moderno. A proposta combina gerenciamento de tarefas, rotina de treinos, alimentação e metas pessoais, tudo dentro de uma estética energética e gamificada.

## Como visualizar

1. Abra o arquivo `index.html` diretamente no navegador ou sirva a pasta com uma extensão de servidor local.
2. Ajuste a largura da janela para algo próximo a 360px para ver o layout mobile conforme o mockup.
3. Use a barra de abas para navegar entre Tarefas, Treino, Alimentação, Metas e Achievements.
4. Utilize os botões “+ Tarefa”, “+ Treino” e “+ Refeição” para registrar novos itens em tempo real.
5. Clique ou use Enter/Espaço nas linhas de treinos e alimentação para marcar o que já foi cumprido.
6. Ajuste os sliders das metas para atualizar o progresso e destravar recompensas e badges na aba de achievements.

## Estrutura

- `index.html` – Estrutura principal da interface do aplicativo.
- `styles.css` – Estilos inspirados no visual escuro com acentos verdes da marca TaskBro.
- `app.js` – Script responsável por abrir os formulários, incluir novos itens e sincronizar progresso das listas, metas e recompensas.

Sinta-se à vontade para evoluir o conceito adicionando novas telas, animações ou integrações.
