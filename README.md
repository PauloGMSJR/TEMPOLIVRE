# TaskBro – Produtividade sem frescura

Interface conceitual do aplicativo TaskBro, um app de produtividade com foco no público masculino moderno. A proposta combina gerenciamento de tarefas, rotina de treinos, alimentação e metas pessoais, tudo dentro de uma estética energética e gamificada.

## Como visualizar

1. Abra o arquivo `index.html` diretamente no navegador, ou utilize uma extensão de servidor local para visualizar em modo responsivo.
2. Ajuste a largura da janela para algo próximo a 360px para ver o layout mobile conforme o mockup.

## Estrutura

- `index.html` – Estrutura principal da interface do aplicativo.
- `styles.css` – Estilos inspirados no visual escuro com acentos verdes da marca TaskBro.

Sinta-se à vontade para evoluir o conceito adicionando novas telas, animações ou integrações.
